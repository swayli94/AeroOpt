# AeroOpt

A multi-objective optimization tool for engineering applications.

This framework is developed to handle the complex needs of
engineering optimization tasks, including:

- calling external tools for evaluation
- parallel evaluation
- pre- and post-processing of the population
- user manipulation of the population

This framework is also developed for other purposes, such as:

- hybridization of different optimization algorithms, machine learning tools, etc.
- adaptive sampling

You can find state of the art single- and multi-objective optimization algorithms
in [pymoo: Multi-objective Optimization in Python](https://pymoo.org/index.html),
where you can also find many more features related to multi-objective optimization
such as visualization and decision making.

## Installation

You can install AeroOpt using pip:

```bash
pip install aeroopt
```
